# NIGHTWIRE — Concept Previsualization

**Status:** Vision review only. **No production development until approval.**

## What this is
An interactive cinematic concept trailer and gameplay previs for a premium neo-noir detective game inspired by 1980s–1990s crime cinema, targeting PC and mobile.

## How to watch
Serve the repo root, then open:

```
http://localhost:8080/nightwire-previs/
```

1. Click **Play Concept Trailer**
2. Use **Next Beat** / `Space` / storyboard scrubber
3. Open the **Vision Deck** for pitch, cast, world, systems, and the approval gate

Demo autopilot: `?demo=1`

## Trailer coverage
Cinematic cameras · crime scene · Detective Vision · inventory · evidence board · dialogue · driving · surveillance · puzzles · day/night · chase · NPC schedules · weather · logo · approval hold

## Approval gate
Reply to approve **Phase 1 — Game Design Document** before any engine project, full GDD, or production systems begin.
