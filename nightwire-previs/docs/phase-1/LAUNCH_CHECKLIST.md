# NIGHTWIRE — Launch Checklist (Phase 1 Draft)

Final execution occurs in Phase 10. This is the living checklist so production never loses the exit criteria.

---

## Content Lock
- [ ] All chapters completable on Standard  
- [ ] All endings reachable and tested  
- [ ] Localization pass (EN complete; others if scoped)  
- [ ] Credits & legal  

## Technical
- [ ] Save/load soak tested  
- [ ] Crash reporters quiet on target devices  
- [ ] Performance tiers validated (High/Mid/Mobile)  
- [ ] Input: KBM, gamepad, touch fully remappable essentials  
- [ ] Accessibility options verified  

## Stores
- [ ] Store page copy + caps + trailer final  
- [ ] Age rating board submissions  
- [ ] Privacy policy / permissions (mic unused; camera in-game only)  
- [ ] Build certification (platform partners as applicable)  

## Ops
- [ ] Day-0 patch pipeline ready  
- [ ] Known issues list  
- [ ] Support contact / FAQ  
- [ ] Analytics events (opt-in) validated  

## Marketing Sync
- [ ] Embargo schedule  
- [ ] Influencer keys  
- [ ] Launch day social kit  
- [ ] Post-launch week-1 content plan  

## Go / No-Go
Requires prod + eng + design + QA sign-off. No launch with critical softlocks or data-loss save bugs.
