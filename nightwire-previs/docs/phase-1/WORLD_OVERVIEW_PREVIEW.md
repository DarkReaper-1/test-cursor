# NIGHTWIRE — World Overview Preview (Phase 1 Seed)

**Full world design = Phase 2.** This locks fantasy geography only.

---

## City: Port Meridian

Coastal industrial city. Rain is weather and metaphor. Neon is commerce and camouflage. Infrastructure is older than the departments that claim to control it.

**Time flavor:** 1989-coded fashion, cars, pagers, CRT precinct — with modern UX clarity for systems.

## Traversal
Semi-open: unlock districts via case pins. Foot + vehicle. Fast travel between safehouses after mid-Act I.

## Time & Weather
Day/night cycle affects schedules and tip expiry. Rain intensity tiers; storms for authored set pieces.

## Environmental storytelling mandate
Every hero space includes: one readable document, one diegetic audio bed cue, one optional vision trace.

Phase 2 delivers maps, POI lists, streaming cuts, and reference boards.
